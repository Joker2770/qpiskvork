Another gomoku manager adapting to Windows and Linux systems.

## summary
This program aims to become a gomoku manager with the [protocol](http://petr.lastovicka.sweb.cz/protocl2en.htm).

## dependencies
dependencies of compiling.
* Qt5+
* qmake or cmake
